
------------------Backend file structure---------
* Relation ship between models

User ────┬─── Cart
         ├─── Order
         ├─── Review
         └─── Coupon Usage

Product ─┬─── Category
         ├─── Review
         ├─── Order Items
         └─── Cart Items

Order ───┬─── User
         ├─── Product (via items)
         └─── Coupon
